parzeltounge
about parzeltounge is a Python library for accessing FramerD knowledge bases and other resources. It is based on a native Python implementation of the DType protocol and attempts to build on the "think and type" (the programmer's equivalent of "look and feel") of the Ptyhon language.
It is currently in very early alpha and is notably missing support for access to file pools and indices and handling of FramerD based inferences. It can be downloaded from SourceForge. Updates may be frequent and otherwise unheralded, so developers are encouraged to monitor the package on SourceForge.
