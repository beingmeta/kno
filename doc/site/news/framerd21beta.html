FramerD 2.1 Beta
  [documentation]  [download]  [demonstrations]  [community] 


FramerD 2.1 is now available as a beta release. Enhancements in FramerD 2.1 include:

  • substantially improved memory performance
  • optimized set/choice operations permitting log or linear time operations in most cases
  • robust and efficient utilization of multiple processors on SMP (symmetric multi processing) architectures
  • better integration with Apache for dynamic generation of web pages with the fdxml executable
  • improved mime parsing with handling of Unicode data in headers

The beta version of FramerD 2.1 can be downloaded from http://www.framerd.org/. Bug reports are welcomed at bug-framerd@framerd.org and a discussion among interested users is hosted at discuss-framerd@framerd.org. (When submitting a bug report, please indicate that you are using FramerD 2.1). Announcements of major releases are available through info-framerd@framerd.org. You may subscribe to these mailing lists from the FramerD web site or by sending email to minordomo@framerd.org with the subject line `subscribe mlist' (e.g. `subscribe discuss-framerd').

Like FramerD 2.0, FramerD 2.1 is released under the GNU GPL (General Public License). Though we have not begun active porting to architectures besides x86 Linux, we do not expect major difficulties. The source for the current release can be downloaded from http://www.framerd.org/ as can precompiled packages for certain architectures.
