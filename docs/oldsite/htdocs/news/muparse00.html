The MUPARSE natural language parser has been released for research and educational purposes.

MUPARSE is a robust free-text natural language (English) parser available for research and educational purposes. MUPARSE is designed for high-throughput natural language processing of unrestricted natural language text. MUPARSE is implemented as a module for FramerD and can be accessed though the FramerD scripting language (FDScript). In addition, MUPARSE can be used as a server accessible through the DTYPE protocol.

MUPARSE is an unprincipled parser which analyzes text using a hand-coded grammar implemented by optimizing finite state machines. This mechanism is used to recognize colocations and assign parts of speech. The output of this process is used to guess at phrase boundaries and futher heuristics make guesses about relations between phrases.

Live demonstrations of MUPARSE are online at http://www.framerd.org/demonstrations.html.
